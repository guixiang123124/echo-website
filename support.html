Frequently Asked Questions

常见问题

Voice input isn't working
Make sure you've granted Microphone permission to Echo. On iOS, go to Settings → Echo → Microphone. On macOS, go to System Settings → Privacy & Security → Microphone.

确认已授予 Echo 麦克风权限。iOS: 设置 → Echo → 麦克风。macOS: 系统设置 → 隐私与安全 → 麦克风。

Keyboard extension doesn't appear
On iOS, go to Settings → General → Keyboard → Keyboards → Add New Keyboard → select Echo. Then tap Echo and enable "Allow Full Access" (required for voice features).

iOS: 设置 → 通用 → 键盘 → 键盘 → 添加新键盘 → 选择 Echo，然后开启"允许完全访问"。

Transcription results are empty or inaccurate
Check that your internet connection is stable — Echo uses cloud-based ASR engines for transcription. Also verify that your API keys are correctly configured in the app settings.

检查网络连接是否稳定，并确认 API Key 已在 App 设置中正确配置。

How do I set up my API keys?
Echo uses a Bring Your Own Key (BYOK) model. You'll need API keys from providers like OpenAI, Deepgram, or Volcano Engine. Go to Echo Settings → API Keys to add them. All keys are stored securely in Apple Keychain.

Echo 采用自带密钥模式。在 Echo 设置 → API Keys 中添加你的密钥，所有密钥安全存储在 Apple Keychain 中。

Does Echo work offline?
Echo requires an internet connection for cloud-based speech recognition and AI polish features. Basic app functionality (viewing history, adjusting settings) works offline.

Echo 的语音识别和 AI 润色功能需要网络连接。查看历史记录、调整设置等基础功能可离线使用。

What platforms does Echo support?
Echo is available on macOS and iOS (iPhone & iPad). Both versions are available on the App Store.

Echo 支持 macOS 和 iOS（iPhone & iPad），均可在 App Store 下载。

How do I cancel my Pro subscription?
Subscriptions are managed through Apple. On iOS: Settings → your name → Subscriptions. On macOS: App Store → your profile → Manage Subscriptions.

订阅通过 Apple 管理。iOS: 设置 → 你的名称 → 订阅。macOS: App Store → 你的头像 → 管理订阅。
